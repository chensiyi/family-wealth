# 历史构成（深化版）

## 导航
- 每个阶段包含：核心意义 → 转变驱动 → 家庭资产影响 → 风险与机会 → 与其他章节关系

## 前现代/封建时代
- 核心意义：土地与赋税为财富核心，权力与秩序决定资产安全
- 转变驱动：宗教与王权的合法性、战争与征服、贸易网络初步形成
- 家庭资产影响：以土地、贵金属与实物为主，传承依赖血缘与地位
- 风险与机会：战争/瘟疫/税赋变动，贸易与城市化带来早期资本积累机会
- 与其他章节关系：宗教与政治力量深度绑定，形成参与者的原型

## 近现代工业化与公司制
- 核心意义：生产力跃迁，公司制与有限责任扩展资本组织能力
- 转变驱动：蒸汽/电力/交通革命，殖民贸易网络与金融工具发展
- 家庭资产影响：从土地到生产性资产与股权，银行体系成为资金中枢
- 风险与机会：周期性过剩与危机、劳资矛盾；产业垄断与规模经济机会
- 与其他章节关系：产业资本与金融资本分工，政治力量介入监管与福利

## 布雷顿森林与美元结算体系
- 核心意义：战后秩序重建，美元成为储备货币，全球贸易与资本流动有序化
- 转变驱动：国际组织与协定确立结算与汇率框架
- 家庭资产影响：跨国配置通道打开，外汇与国际托管重要性提升
- 风险与机会：汇率制度调整与资本账户管理，出口型经济的结构性红利
- 与其他章节关系：主权层规则明确，金融资本全球化

## 金本位退出与信用货币时代
- 核心意义：记账单位与购买力脱钩，央行与银行体系主导信用扩张与收缩
- 转变驱动：利率政策、财政赤字与金融创新，形成资产价格的金融化
- 家庭资产影响：通胀与资产价格共振，现金购买力长期受侵蚀
- 风险与机会：紧缩期清算与失业风险；扩张期权益与资源类资产受益
- 与其他章节关系：信用周期成为经济生态的主线，策略需以现金流与结构对冲

## 金融化与影子银行
- 核心意义：表外信用创造与复杂结构化产品，风险在系统内隐蔽迁移
- 转变驱动：监管套利、创新产品与全球资本流动
- 家庭资产影响：收益表面提升，实际风险被转移与放大
- 风险与机会：流动性枯竭与连锁违约；合规透明的托管与风险隔离更重要
- 与其他章节关系：托管与对手方风险成为家庭架构设计的核心

## 技术平台化与数据金融
- 核心意义：支付与数据基础设施重塑信用评估与交易，软权力增强
- 转变驱动：移动互联网、云与AI、平台网络效应
- 家庭资产影响：账户与数据安全成为资产安全的先决，平台迁移风险上升
- 风险与机会：平台政策/产品下线造成断崖式风险；数字化降低交易成本与信息不对称
- 与其他章节关系：机制与策略加入平台维度，极端事件预案必须包含数据与接口

## 中国与新兴市场路径（特例）
- 核心意义：双轨制与渐进开放，监管与市场并存形成独特套利与风险结构
- 转变驱动：城市化与地方财政、平台经济崛起
- 家庭资产影响：房地产与平台资产的周期性重要性，资本账户与汇率管理的影响
- 风险与机会：监管突变与账户可达性风险；制造与供应链重构的长期机会
- 与其他章节关系：在岸/离岸配比与法域选择成为策略的关键参数

## 串联逻辑（从秩序到家庭策略）
- 秩序形成 → 参与者博弈 → 规则执行 → 信用与价格 → 托管与迁移 → 家庭架构与行动
- 每一次范式变化都改变现金流质量与资产持有形态，策略应优先结构与合规
