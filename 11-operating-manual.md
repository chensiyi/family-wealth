# 系统运行说明书：操作手册 (The Operating Manual)

> **文件摘要**: 本章提供家族财富管理系统的完整操作手册，整合情报系统与行为体系形成闭环运行机制。通过角色分工、运行节奏和关键流程的标准化设计，确保复杂投资决策能够像精密机器般可靠执行，并建立持续优化的反馈迭代机制。

## 1. 系统概述
本手册旨在将“家族资产打理”知识体系转化为可执行的日常操作。它集成了**情报系统（Eyes）**与**行为体系（Hands）**，确保在复杂的外部环境下，家族财富管理能够像精密机器一样运转。

### 核心循环
1.  **输入**：市场数据、新闻、财报（经过情报系统过滤）。
2.  **处理**：沙盘推演、底牌透视、策略匹配。
3.  **输出**：交易指令、风控动作、资产再平衡。
4.  **反馈**：结果复盘、系统迭代。

---

## 2. 角色与权限

*   **情报官 (IO)**：负责信息的收集、清洗与初步研判（L1-L2）。
*   **指挥官 (CO)**：负责查看沙盘、制定决策（L3-L4）并下达指令。
*   **执行官 (Ops)**：负责具体交易操作、资金划转与记录（SOP执行）。
*   **审计员 (Audit)**：负责事后核查、红队测试与合规监督。
*   *注：在小家庭中，这些角色可能由1-2人兼任，但思维模式必须切换。*

### 家族办公室特有的角色设置

**家族治理委员会**：
- **组成**：家族长辈、专业顾问、年轻一代代表
- **职责**：制定家族宪章、传承规划、重大投资决策审批
- **频率**：年度全体会议 + 季度核心成员会议

**投资决策委员会**：
- **组成**：首席投资官、外部投资顾问、家族代表
- **职责**：具体投资标的筛选、风险评估、组合调整
- **频率**：月度例会 + 重大事项临时会议

**合规与风控委员会**：
- **组成**：首席合规官、法律顾问、税务专家
- **职责**：监管合规审查、风险监控、应急预案制定
- **频率**：季度审查 + 不定期专项检查

## 3. 日常运行节奏 (The Routine)

### 每日 (Daily) - *耗时：15-30分钟*
*   **情报官**：
    *   浏览"宏观气象站"关键指标（利率、汇率）。
    *   查阅"红旗侦测雷达"是否有报警。
    *   快速扫描核心资产价格波动。
*   **指挥官**：
    *   若无红旗/无重大波动 -> **无动作（Do Nothing）**。
    *   若有触发信号 -> 启动"OODA循环"。

### 信用货币环境下的特殊监控机制

**月度宏观经济评估**：
- **货币供应量变化**：M1、M2增速与历史水平对比
- **信贷扩张速度**：银行放贷规模及结构变化
- **通胀预期指标**：breakeven inflation rate、消费者预期调查
- **央行沟通分析**：议息会议纪要、官员讲话解读

**季度投资组合压力测试**：
- **不同通胀 scenario**下的资产表现模拟
- **流动性紧缩环境**中的资产变现能力评估
- **汇率大幅波动**对海外资产的影响分析
- **政策转向风险**对各类资产的冲击测试

**年度战略回顾与调整**：
- **宏观环境变化评估**：过去一年的主要变化及其影响
- **投资策略有效性检验**：是否达到预期风险调整后收益
- **家族目标适应性检查**：投资策略是否仍符合家族长期目标
- **治理体系优化**：根据实践经验调整决策流程和权限设置

### 每周 (Weekly) - *耗时：1小时*
*   **执行官**：
    *   更新家庭资产负债表（粗略版）。
    *   记录本周现金流变动（收入/支出）。
*   **指挥官**：
    *   回顾本周市场热点，判断是否属于“噪音”。
    *   阅读一篇深度研报或经典文献（知识库建设）。

### 每月 (Monthly) - *耗时：2-3小时*
*   **全员**：
    *   **月度情报会**：汇总L3级别情报，更新战场沙盘。
    *   **底牌透视**：对持仓的一项资产进行深度体检（轮询制）。
    *   **执行检查**：核对交易日志与银行/券商对账单。

### 每季度 (Quarterly) - *耗时：半天*
*   **全员**：
    *   **再平衡会议 (SOP-02)**：对比目标配置，执行买卖操作。
    *   **绩效归因**：赚的是运气的钱还是逻辑的钱？
    *   **系统升级**：更新白名单/黑名单，修订SOP。

#### 季度会议议程模板
- 目标函数检查：本季目标是否达成，偏差原因
- 权重与久期：对比目标中枢与当前持仓，拟议调整
- 信号核对：查看“信号—阈值—动作表”（见 [08 周期配置](08-cycle-configuration.md)）
- 红旗与风控：本季红旗事件与处置记录，新增防线
- SOP修订：新增/废止流程，触发器与步骤更新
- 合规与税务：重要申报与审计节点确认
- 行动项与负责人：明确截止日期与记录位置

### 每年 (Yearly) - *耗时：1-2天*
*   **战略复盘**：
    *   回顾家族目标（定位与意义章节）。
    *   压力测试演练（SOP-03 紧急熔断）。
    *   税务与合规年度体检。
    *   制定下一年度的大类资产配置中枢。

## 4. 关键流程图解

### 流程 A：新投资决策
1.  **情报输入** -> 发现潜在机会（如：某地房产/某类债券）。
2.  **初筛** -> 检查是否在“白名单”内？是否违反“黑名单”？
3.  **底牌透视** -> 启动 `templates/source-and-fraud-checklist.md`。
    *   利益结构？极端风险？法律权属？
4.  **沙盘推演** -> 放入组合中模拟，对整体风险有何影响？
5.  **决策 (CO)** -> 批准/否决/需更多信息。
6.  **执行 (Ops)** -> 执行 SOP-01（分批建仓）。
7.  **归档** -> 存入内部维基。

### 流程 B：危机应对
1.  **红旗报警** -> 情报系统监测到重大风险（如：托管行信用违约互换暴涨）。
2.  **紧急会议** -> 确认情报真实性（L2验证）。
3.  **预案激活** -> 调出 SOP-03（紧急熔断）。
4.  **执行撤退** -> 变现、转账、备份。
5.  **静默观察** -> 在安全地带等待局势明朗。

## 5. 系统维护与迭代

*   **规则优于预测**：当市场走势与你的预测不符时，遵守预设的规则（止损/再平衡），而不是修正你的预测。
*   **记录就是一切**：没有记录的决策是无法复盘的。保持诚实，记录下当时的恐惧与贪婪。
*   **保持简单**：如果系统过于复杂导致难以坚持，那就简化它。一个能运行的简单系统胜过一个瘫痪的完美系统。
