- 标题
- 来源与链接
- 时间与法域
- 概要与要点
- 相关风险/机会

## 更新节奏与标注
- 每次更新记录日期与版本号（示例：v2026.01.29）
- 在 README 与各章节顶部维护「最近更新」小结
- 重大变更撰写变更日志：变更原因、影响范围、后续动作

## 引用与合规
- 保留原始出处与快照，避免断链与误读
- 对涉及法律/税务的条目，标注非建议与需咨询专业人士
- 对涉及平台与技术的条目，记录版本与适用范围
