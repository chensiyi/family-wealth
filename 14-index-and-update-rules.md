## 🎯 索引结构重构：按决策场景划分

### [平时] 资产维护索引
**核心任务**：保持系统稳定运行，寻找廉价资金机会

#### 资金成本监控表
- 各法域托管行实时利率表（寻找最便宜的资金来源）
- 境外私人银行质押利率对比（Libor+1% 至 Libor+2% 区间）
- 数据资产信用背书抵押机会清单

#### 现金流管理
- 第二桶金（收租）现金流到账确认清单
- 干粉储备监控（维持20%纯现金比例）
- 廉价资金通道维护状态表

### [战时] 狩猎与防御索引
**核心任务**：系统性危机中的机会捕捉和风险规避

#### 紧急响应手册
- SOP-03 紧急跳船手册（物理备份位置、离线转账授权）
- 核心资产重置成本数据库（用于快速判定标的是否真的便宜）
- 法域切换应急预案（在岸→离岸的快速通道）

### 状态切换卡：全天候 ↔ 狙击模式

#### 全天候健康度检查表（月度）
- [ ] 资产配置偏离度 < 5%（股/债/商品/黄金平衡）
- [ ] 法域分布均衡性检查（避免单一法域风险）
- [ ] 现金流充足性（维持3-6个月生活支出）
- [ ] 廉价资金通道成本稳定性（Libor+1.2%以内）

#### 狙击标的预案库（动态维护）
**核心原则**：列出3-5个"如果跌到这个价，我就用廉价资金接手"的核心资产

##### 商业地产类
- [ ] 核心地段商办底层资产（重置成本vs现价比率监控）
- [ ] 优质物流仓储设施（通胀对冲属性）

##### 股权类
- [ ] 硬核科技公司清算债权（巨鲸溃败信号监测）
- [ ] 具备护城河的消费品牌股权（品牌价值锚定）

##### 数据资产类
- [ ] 核心数据分发权（05.2章数字锚定资产）
- [ ] 行业know-how数据库（稀缺性价值评估）

#### 切换触发条件
**从全天候→狙击模式**：
- 重置成本雷达显示目标资产价格 < 重置成本 × 50%
- 巨鲸溃败信号连续出现（L2.5玩家大规模平仓）
- 信贷脉冲转负且传导失效超过30天

**从狙击→全天候模式**：
- 狙击仓位达到预定收益目标（通常为50-100%）
- 系统性风险指标恢复正常
- 干粉储备降至安全线以下（<15%）

## 📋 条目模板升级

### 基础模板
- **标题**：明确标识内容主题
- **来源与链接**：保留原始出处，避免断链
- **时间与法域**：标注时效性和适用范围
- **概要与要点**：提炼核心信息
- **相关风险/机会**：平衡视角分析
- **与家族策略的关系**：明确战略定位

### 单兵作战专用模板
- **资金成本**：当前融资成本和替代方案
- **流动性评估**：资产变现能力和时间窗口
- **法域风险**：所在法域的政治经济稳定性
- **执行门槛**：操作复杂度和所需资源
- **生存价值**：在极端情况下的保值能力

## ⚙️ 更新规则：引入压力测试迭代

### 触发机制
- **气象突变**：08章定义的重大政策变化、基准利率暴涨、地缘风险升级
- **资金成本上升**：廉价资金通道成本超过预设阈值
- **法域风险升级**：所在法域出现重大不利变化

### 更新流程
1. **24小时内响应**：更新流动性出口有效性
2. **季度复盘**：审计融资成本和替代方案
3. **年度压力测试**：模拟极端情况下的生存能力

### 版本管理
- 每次更新记录日期与版本号（示例：v2026.02.12-SOLO）
- 在README与各章节顶部维护「最近更新」小结
- 重大变更撰写变更日志：变更原因、影响范围、后续动作

## 🛡️ 1亿量级生存审计表

### 核心生存指标
- [ ] 是否拥有随时可以调用的20%纯现金（干粉）？
- [ ] 第一桶金是否实现了与个人身份的法律隔离？
- [ ] 如果明天所有在岸账户被冻结，离岸廉价资金通道能否支撑生活并启动狩猎？
- [ ] 是否建立了跨法域的"热切换"能力？

### 机会捕捉能力
- [ ] 是否建立了巨鲸溃败信号的监测机制？
- [ ] 是否掌握了核心资产的重置成本数据？
- [ ] 是否与顶级玩家（L2.5）建立了信息共享渠道？
- [ ] 是否具备快速调头和精准伏击的执行能力？

## 📚 引用与合规

### 信息源管理
- 保留原始出处与快照，避免断链与误读
- 对涉及法律/税务的条目，标注非建议与需咨询专业人士
- 对涉及平台与技术的条目，记录版本与适用范围

### 风险提示
- 所有投资建议均基于理论分析，不构成具体投资建议
- 跨法域操作需充分了解当地法律法规
- 廉价资金使用需严格控制杠杆比例（建议LTV不超过30%）
