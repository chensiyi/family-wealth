# 游戏机制与策略（经济生态刻画）

## 生态隐喻
- 能量与资源：现金流与信用就是能量，资产是栖息地
- 捕食与寄生：税费与通胀是自然消耗；不透明的结构化产品与高费用载体是寄生体
- 迁徙与栖息：资本跨法域迁徙，平台与托管的生态选择决定生存概率
- 繁殖与凋亡：信用扩张促繁殖，清算与去杠杆导致凋亡

## 关键运行过程
- 信用创造链路：央行基础货币 → 银行贷款与表外信用 → 资产价格 → 抵押再融资 → 周期强化
- 清算瀑布：利率上升/现金流恶化 → 融资收紧 → 被动卖出与连锁违约 → 价格与流动性双杀
- 监管窗口：规则名义不变但执行窗口变化 → 产品与渠道的可用性差异化
- 平台迁移：产品下线/API调整/账户冻结 → 数据与资产迁移成本与失败风险
- 外汇与通胀：记账单位变化掩盖真实购买力 → 配置需区分名义收益与实质收益

## 策略映射
- 结构优先：公司/信托/保险载体构建风险隔离，先保栖息地再谈繁殖
- 现金流锚定：以稳健现金流覆盖生活与教育，建立滚动应急金与备用信用额度
- 多元化与分层：在岸/离岸、显性/隐性、权益/防御、平台冗余的组合
- 合规优化：居住地与税务居民身份规划、合法税务优化与合规清单
- 托管与对手方：稳健机构与透明度要求，多通道冗余防单点失败
- 极端事件预案：账户冻结与平台迁移剧本、通讯与文件备份、跨法域取用

## 场景化叙事（示例）
- 扩张季：低利率→信用扩张→资产价格上涨→杠杆增加；策略：控制杠杆、锁定长期资金、审慎增配生产性与资源类资产
- 收缩季：利率上升→融资收紧→现金流压力→清算瀑布；策略：提升现金流质量、增加防御性资产、减少非核心敞口
- 平台迁徙：账户受限→数据出口→新平台接入与测试→回滚机制；策略：保持两家以上托管与备份账户，定期演练
- 地缘突变：制裁/资本管制→跨法域取用受限；策略：降低可冻结资产比例、身份与法域冗余、保留实体现金与必需品通道
