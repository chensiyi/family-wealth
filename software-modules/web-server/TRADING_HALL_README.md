# 实时交易大厅使用说明

## 📋 系统概述

实时交易大厅是一个专业的股票交易模拟平台，为家族财富管理系统提供完整的虚拟交易环境。系统集成了真实的市场数据模拟、专业级交易功能和全面的风险管理工具。

## 🚀 快速启动

### 1. 启动后端服务
```bash
cd software-modules/web-server
python trading_hall_api.py
```

### 2. 启动前端服务
```bash
cd software-modules/web-server
npm start
```

### 3. 访问交易大厅
打开浏览器访问: http://localhost:3000/trading-hall

## 🎯 核心功能

### 股票浏览与搜索
- **实时行情**: 显示主要股票的最新价格、涨跌幅、成交量
- **智能搜索**: 支持股票代码和名称模糊搜索
- **热门股票**: 展示市场关注度较高的标的

### 专业交易功能
- **多种订单类型**: 
  - 限价单：指定价格买卖
  - 市价单：按当前市场价格立即成交
  - 止损单：设置触发价格自动买卖
- **完整交易流程**: 从下单到成交的全流程模拟
- **资金安全**: 自动验证资金充足性和持仓可用性

### 持仓管理
- **实时盈亏**: 动态计算未实现盈亏和收益率
- **成本分析**: 平均持股成本和市场价值对比
- **风险监控**: 持仓集中度和风险敞口分析

### 订单管理
- **委托队列**: 实时查看所有未成交订单
- **一键撤销**: 支持单笔或批量撤销订单
- **成交记录**: 完整的交易历史追溯

## 🎮 使用示例

### 基本交易流程
1. 在左侧股票列表中选择目标股票
2. 在交易面板设置买卖方向、数量和价格
3. 点击"提交订单"完成下单
4. 在"委托订单"标签页查看订单状态
5. 成交后可在"我的持仓"查看最新持仓情况

### 高级功能
- 使用市价单快速成交
- 设置止损单控制风险
- 通过买卖盘分析市场情绪
- 查看历史交易记录优化策略

## ⚠️ 注意事项

- 本系统为模拟交易平台，不涉及真实资金
- 所有数据均为模拟生成，仅供参考
- 交易结果不代表真实市场表现
- 建议结合实际市场情况进行决策

## 🆘 技术支持

如遇问题，请检查：

2. 前端Web服务是否正常运行（端口3000）
3. 浏览器控制台是否有错误信息
